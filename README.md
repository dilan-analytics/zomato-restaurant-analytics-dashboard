# 🍽️ Zomato Restaurant Analytics Dashboard

## 📌 Project Overview

This project analyzes restaurant data to understand ratings, cost distribution, and online ordering trends using an interactive Excel dashboard.

---

## 📂 Dataset

* data/raw_data.csv
* data/cleaned_data.csv

---

## 📊 Dashboard Features

* Rating Distribution
* Market Share by Category
* Cost Distribution
* Ratings vs Votes
* Top Restaurants by Votes

---

## 🎛️ Interactivity

* Slicers:

  * Online Order
  * Restaurant Type(listed_in_type)
  * Cost Category

---

## 🛠️ Tools Used

* Python
* Excel
* SQL


---
## 📓 Jupyter Notebook

* notebooks/zomato_analysis.ipynb

This notebook includes:

* Data cleaning using Python
* Handling missing values
* Feature engineering
* Exploratory data analysis (EDA)
